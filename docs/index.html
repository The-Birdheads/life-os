アプリについて

Habitas（ハビタス）は、あなたの毎日をより豊かに、より計画的にするためのシンプルなライフログ・タスク管理アプリです。 「毎日続けたい習慣」「今日終わらせたいタスク」「突発的に起こった行動」の3つを1つのタイムラインでシームレスに記録・管理できます。
