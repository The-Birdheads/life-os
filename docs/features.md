# Habitas (LifeOS) 実装済み機能一覧

リリース前のテスト用に、現在実装されている主な機能の一覧をまとめました。

| カテゴリ | 機能名 | 概要（ユーザーができること） | 関連データ/API | 状態 |
| :--- | :--- | :--- | :--- | :--- |
| **認証** | ログイン/ログアウト | Googleアカウントを用いたログインおよびログアウトができる | Supabase Auth | 実装済 |
| **認証** | ディープリンクログイン | iOSなどネイティブアプリからOAuthログイン後、アプリに復帰してログイン状態になる | Capacitor App (`appUrlOpen`) | 実装済 |
| **データ管理** | クラウドデータ同期 | ローカル(未ログイン)のデータをクラウドへ移行し、複数端末間で同期する | SQLite (JeepSqlite) / Supabase | 実装済 |
| **タスク・行動管理** | タスク(Oneoff)の管理 | 単発タスクの登録・編集・削除・完了チェックができる（期限・優先度・ボリューム指定可） | `tasks`, `task_entries` | 実装済 |
| **タスク・行動管理** | 習慣(Habit)の管理 | 繰り返し行う習慣の登録・編集・削除・完了チェックができる | `tasks`, `task_entries` | 実装済 |
| **タスク・行動管理** | 行動(Action)の記録 | 日々の具体的な行動ログを記録・編集・削除できる（任意メモ、ボリューム記録可） | `actions`, `action_entries` | 実装済 |
| **ダッシュボード** | Today表示 | 「今日」のタスクや習慣、行動を確認し、フィルタ機能で絞り込み表示できる | TodayViewコンポーネント | 実装済 |
| **振り返り** | 日日の振り返り記録 | その日の充実度（1〜100）と振り返りメモを入力して保存できる | `daily_logs` | 実装済 |
| **振り返り** | 実績のサマリダッシュボード | その日の完了した習慣・タスク・行動の数と合計ボリューム(Vol)を確認できる | ReviewViewコンポーネント | 実装済 |
| **統計データ** | 週間データの閲覧 | 過去7日間の平均充実度、および各日のタスク・習慣・行動の実施数を一覧で確認できる | WeekViewコンポーネント | 実装済 |
| **設定・通知** | 通知スケジュール管理 | アプリからローカル通知（リマインダーなど）のスケジュール登録および通知タップ時の画面遷移ができる | Capacitor Local Notifications | 実装済 |
| **収益化** | バナー広告表示 | 画面下部にAdMobによるバナー広告を表示する | Capacitor Community AdMob | 実装済 |

※ SQLiteを用いたオフラインファーストのアプローチとなっており、基本操作(`upsert`や`delete`など)は全てローカルデータベースに保存された後、同期処理(`sync`)を通じてSupabaseへ反映される設計になっています。
